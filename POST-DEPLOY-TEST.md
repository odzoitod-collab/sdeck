## Полное тестирование

### Шаг 1: Telegram Bot
1. Напишите боту `/start`
2. Создайте товар через "➕ Создать товар"
3. Заполните все 5 шагов (включая цену доставки)
4. Получите ссылку на товар

### Шаг 2: Web Interface
1. Откройте ссылку на товар
2. Заполните форму получателя
3. Перейдите к оплате
4. Загрузите тестовый скриншот
5. Подтвердите оплату

### Шаг 3: Telegram Channel
1. Проверьте канал `-1003631337137`
2. Убедитесь, что пришло сообщение с:
   - Кодом объявления
   - Никнеймом продавца
   - Информацией о товаре
   - Скриншотом оплаты

## Возможные проблемы

### API не работает
- Проверьте переменные окружения в Vercel
- Убедитесь, что `@vercel/node` установлен
- Проверьте логи функций в Vercel Dashboard

### Telegram не получает сообщения
- Проверьте права бота в канале
- Убедитесь, что Channel ID правильный: `-1003631337137`
- Проверьте Bot Token: `7963734070:AAG5i2NL52UDanuqC7kRFNdZMXfapQG1c3E`

### Supabase ошибки
- Проверьте подключение к базе данных
- Убедитесь, что таблица `orders` содержит поле `seller_username`
- Проверьте RLS политики
