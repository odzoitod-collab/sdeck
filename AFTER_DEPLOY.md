```

```
- Должна загрузиться главная страница
- Форма поиска заказа должна работать

### 2. Проверьте API endpoint
```
https://sdeck.vercel.app/api/send-to-telegram
```
- Должен вернуть ошибку 405 (Method not allowed) при GET запросе
- Это нормально - endpoint принимает только POST

## 🐛 Возможные проблемы

### API не отправляет в Telegram

**Проверьте:**
1. Бот добавлен в канал `-1003631337137` как администратор
2. Bot Token правильный: `7963734070:AAG5i2NL52UDanuqC7kRFNdZMXfapQG1c3E`
3. Channel ID правильный: `-1003631337137`

**Решение:**
- Проверьте логи функции в Vercel
- Убедитесь, что бот имеет права на отправку сообщений

### Заказы не загружаются

**Проверьте:**
1. Переменные окружения в Vercel Dashboard
2. Подключение к Supabase
3. Таблица `orders` существует и содержит данные

**Решение:**
- Проверьте Supabase Dashboard
- Убедитесь, что RLS политики настроены правильно

### Скриншоты не загружаются

**Проверьте:**
1. Размер файла (максимум 5MB)
2. Формат файла (JPG, PNG)
3. Логи API функции
